# 🚦 Smart Traffic Management System - BMC Website

This project is a Business Model Canvas (BMC) website for a Smart Traffic Management System.

## 📌 Features
- Clean UI design
- Responsive layout
- Hover animation
- Easy GitHub hosting

## 🚀 How to Run
1. Download project
2. Open index.html in browser

## 🌐 GitHub Pages Deployment
1. Upload files to GitHub repo
2. Go to Settings → Pages
3. Select main branch
4. Done ✅
